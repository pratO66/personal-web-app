THE FOUR OFFICIAL UI STYLES
From the UI Art Bible (Part 1 — behance.net/gallery/118663901). Each style maps to a Night City societal era. Only Neo Militarism applies to the resume site.
"This is the initial form of the User Interface Art Bible, the base of an aesthetical guide through the historical and technological eras of Cyberpunk 2077. Design and civilization have danced hand-in-hand since time immemorial. Whatever the human activity, design has never failed to react." — VLADIMÍR VILIMOVSKÝ, UI Art Bible Foreword (Part 1)
STYLE 01
ENTROPISM

Functional poverty. CRT terminal aesthetic, worn surfaces. Born from scarcity — minimal resources expressed through minimal design. Phosphor-green data readouts, industrial grime.

// Survival over style
Use in: slum screens, scav gear, lower-city terminals
STYLE 02
KITSCH

Commercial excess. Maximum brightness, maximum noise. Neon signage and advertising saturation. Over-the-top visual overload — every surface screaming for attention.

// Louder. More. Always more.
Use in: street ads, braindance clubs, fast-food kiosks
STYLE 03 — DEFAULT PLAYER UI
NEO MILITARISM

Cold, threatening elegance. Corporate precision with militaristic formality. "Substance over style." Forceful, functional, high-quality — never flashy. RED is the primary color — a deliberate non-conformist decision against all UI conventions. Encountered in wealthy corporate zones and business districts.

// Substance over style.
⚠ IMPLEMENT THIS STYLE. All other styles are reference only.
STYLE 04
NEO KITSCH

Premium commercial luxury. Gold accents, high-end advertising aesthetic. Status signaling through material quality. Refined excess — not cheap neon but opulent gold. Corporate elite.

// Luxury as language.
Use in: corpo elite panels, high-end vendor interfaces
"The biggest struggle was with the efforts to create the military system, with the hostile nature and with a degree of aggression expressed and rendered in RED colour. It was a bold decision and a walk against the stream of all the conventions and design habits. Just because the colour RED is a default it is inevitably causing a discrepancy with its general use, communicating malfunctioning, warnings, low battery, danger, etc. The aesthetical direction was based on UI Art Bible and we kept on it — we were iterating and exploring ways around to make our UI comfortable to use and to navigate through, yet having the non-conformist attitude. That was our goal with UI for Cyberpunk 2077. Note: There were a few side steps with different colour experiments, but it never really felt right." — VLADIMÍR VILIMOVSKÝ, Part 2 Introduction (behance.net/gallery/133185623)

CRITICAL DESIGN RULE
RED (#C5003C) IS THE DEFAULT PRIMARY COLOR — NOT AN ERROR STATE.
In Neo Militarism, red communicates active selection, primary hierarchy, and player identity. Do NOT remap red exclusively to errors or danger. Use --yellow (#F3E600) for actual warnings. Use --teal (#55EAD4) for hacking/success states. The non-conformist use of red as primary is the defining signature of this UI system.

ANIMATION — OFFICIAL PATTERNS
Documented in Part 2: "schematically showing why our HUD is curved by the shader." Glitch + flicker = signature effects. HUD has viewport-curve perspective transform.
// GLITCH TEXT (HERO NAME)
/* ::before = red offset, ::after = teal offset */
.cp-glitch { position: relative; }
.cp-glitch::before {
  content: attr(data-text);
  position: absolute; inset: 0;
  color: #FF003C;     /* red offset layer */
  animation: glitch-red 6s steps(2) infinite;
  clip-path: polygon(0 20%, 100% 20%, 100% 40%, 0 40%);
}
.cp-glitch::after {
  content: attr(data-text);
  color: #55EAD4;     /* teal offset layer */
  animation: glitch-teal 6s steps(2) infinite 0.1s;
  clip-path: polygon(0 60%, 100% 60%, 100% 75%, 0 75%);
}
@keyframes glitch-red {
  0%,88%,100% { transform: translate(0); opacity: 0; }
  89%  { transform: translate(-3px,1px);  opacity: 0.8; }
  91%  { transform: translate(3px,-1px);  opacity: 0.8; }
  93%  { transform: translate(-2px,2px); opacity: 0.6; }
  97%  { transform: translate(0);         opacity: 0; }
}
// SCREEN FLICKER + PULSE GLOW
/* Ambient screen flicker — full page */

  0%,100%  { opacity: 1;    }
  92%      { opacity: 1;    }
  93%      { opacity: 0.85; }
  94%      { opacity: 1;    }
  96.5%    { opacity: 0.9;  }
  97%      { opacity: 1;    }
}

  0%,100% {
    box-shadow: 0 0 0 1px #C5003C,
